Comandos para git

  1. Primer comando: git init (para iniciar git 1 sola vez por proyecto)
  2. Segundo comando: git status (para chequear status en nuestro proyecto)
  3. Tercer comando: git add . (para trackear todos los archivos)
  4. Cuarto comando: git commit -m "mensaje" (para hacer el paquete de una version para subirla a la nube)
